# Gameplay e Mecânicas

## Sistemas Principais

### 1. Sistema de Dados
- Representam a "alma" do jogador
- Evolução de d6 até d20
- Mecânica de escolha estratégica

### 2. Sistema de Cartas
- Efeitos passivos que modificam mecânicas
- Aquisição através de derrota de inimigos
