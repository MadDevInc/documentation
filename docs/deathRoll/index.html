Death Roll

Sobre o Jogo

Death Roll é um jogo de terror e estratégia inspirado em jogos como Buckshot Roulette, Inscryption e Balatro. Jogue como uma alma condenada que deve lutar para se salvar das garras de criaturas do submundo.
